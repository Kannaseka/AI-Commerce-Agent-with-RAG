Professional AI-powered chat assistant for WooCommerce stores. Handles product queries, order tracking, and customer support automatically.

== Description ==

**Transform Your Customer Service with AI**

Roze AI Chat Assistant is a cutting-edge conversational AI that integrates seamlessly with your WooCommerce store. It provides instant, accurate responses to customer queries 24/7.

= Features =

* ✅ **Live Product Information** - Real-time product prices, stock status, and descriptions
* ✅ **Order Tracking** - Automatic order status lookups
* ✅ **Knowledge Base Integration** - Answers from your website content, blogs, and policies
* ✅ **Intelligent Fallback** - Never shows technical errors to customers
* ✅ **Response Caching** - Lightning-fast responses for common questions
* ✅ **Analytics Dashboard** - Track conversations and popular questions
* ✅ **Fully Customizable** - Colors, position, welcome message, and more
* ✅ **Mobile Responsive** - Perfect experience on all devices
* ✅ **Business Hours** - Configure operating hours and after-hours messages

= How It Works =

1. Install and activate the plugin
2. Configure your AI backend endpoint
3. Customize appearance and behavior
4. Go live - the chat widget appears automatically!

= Requirements =

* WordPress 5.0 or higher
* WooCommerce (for product integration)
* PHP 7.4 or higher
* AI Backend running (FastAPI + Python)



== Installation ==

1. Upload the `roze-ai-chat` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to AI Chat in the admin menu
4. Configure your API endpoint and preferences
5. Save changes and test!

== Frequently Asked Questions ==

= Do I need coding skills? =

No! The plugin is designed for non-technical users. Simply install, configure the endpoint, and you're done.

= What AI models are supported? =

We support Groq (Llama 3.3 70B) and OpenAI (GPT-4o Mini). Groq is recommended for production due to speed and cost.

= Can I customize the appearance? =

Yes! Use the Advanced Settings Dashboard to customize colors, position, welcome message, and more.

= Does it work with WooCommerce? =

Yes! The AI automatically pulls live product data, pricing, and order information from your WooCommerce store.

= Is it mobile-friendly? =

Absolutely! The chat widget is fully responsive and works perfectly on mobile devices.
